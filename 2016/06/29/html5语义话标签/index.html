html5语义话标签

header 头部

footer 尾部

section 标签定义文档中的节（section、区段）。比如章节、页眉、页脚或文档中的其他部分。

aside 文章的侧栏

nav 导航

article 规定独立的自包含内容，可以包裹section

hgroup 包裹多个h1，h2…

mark 标记 高亮文本常作用于搜素高亮

meter 定义度量衡表示在一定范围内的值，如投票占比，使用量，游戏里的血条等 属性有value配min和max optimum配high和low

progress 定义进度 进度条 属性有value配max和min

time 定义事件和日期 datetime=“YYYY-MM-DDThh:mm:ssTZD”

wbr 定义换行

abbr 定义缩写

cite blockquote q 都是引入的意思 第一个斜体第二个有margin第三个有引号

output 输出 该标签要搭配form和input使用

code 格式化代码

pre 定义预格式化的文本 在pre元素中的文本通常会保留空格和换行符。而文本也会呈现为等宽字体

detail 定义元素的细节 搭配summary（定义细节标题）使用，属性open=“open” summary应为details元素的第一个子元素

dialog 定义对话框 open属性

figure 图表 figcaption 定义图标标题

menu 菜单

main 主要内容

map 用于客户端图像映射，图像映射指带有可点击区域的一幅图像 搭配usemap和area标签使用

address 一般用于footer的地址和电话信息栏非友情链接

下面都是input的属性值

tel 用于输入电话号码

search 用于提示用户输入要搜素的文字 search和text质检的区别主要在于样式上

url 用于输入单个网址

email 用于输入单个电子邮件地址 如果制定了multiple属性，就可以输入多个电子邮件地址，以逗号分隔

datetime 用于输入时区设置韦UTC的日期和时间

date 用于输入不含时区的日期

month 用于输入含年份和月份但不含时区的日期

week 用于输入含年份和周数但不含时区的日期

time 用于输入含小时，分钟，秒和秒的小数部分但不含时区的时间值

datetime-local 用于输入不含时区的日期和时间

number 用户输入数字

range 用于输入数字，但与number的区别在于无需输入具体数字 实施形式大部分为滑块

color 用于通过颜色池控件选择颜色

label 为 input 元素定义标注 搭配for使用 意识就是说可以点框前面的文字选定后面的

datailst 定义选项列表 配合input使用 list属性配合id名

ins 定义已经被插入文档中得文本 del为删除

samp 定义计算机程序的样本文本

上面都是input的属性值

表单属性

autofocus 用于在网页加载后对焦到相关元素上德输入 它的目标可以是输入。选择，文本区域和按钮

placeholder 用于提示用户应输入的数据类型 在对焦到相关元素以及用户输入数据之前，系统会以浅色文字显示占位符值。您可以在输入和文本区域中指定该值。

required 如果设置该属性，它规定用户在提交表单之前必须选择一个值

pattern 属性规定用于验证输入字段的模式 用于身份证号码类型的

autocomplete 自动完成允许浏览器预测对字段的输入。当用户在字段开始键入时，浏览器基于之前键入过的值，应该显示出在字段中填写的选项

aduio

autoplay 自动播放

controls 属性规定浏览器应该为视频提供播放控件

loop 单曲循环

preload 提前加载

src 地址

source 单表签 title属性

track 定义用在媒体播放器中得文本轨道 歌词文件 字幕文件

video

上同

width 设置宽度
