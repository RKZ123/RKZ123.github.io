Transition

transition 过渡

transition：all 0.5s ease-in-out 1s；

上面四个属性为：

all： transition-property执行过渡变化的属性 可以是width heigh等

0.5s：transition-duration过渡变换延续的时间

ease-in-out：transition-time-function过渡变换的速率效果方式

1s：transition-delay换延迟时间

transform2D 变形

transform：rotate 旋转 | scale 缩放 | skew 扭曲 | translate 移动 | matrix 矩阵变形

rotate 旋转

通过指定的角度参数对原元素指定一个2D rotation（2D旋转）

deg可为负数

rotateX：元素围绕其X轴以给定的读书进行旋转 上下旋转 顺时针为正

rotateY：元素围绕其Y轴以给定的读书进行旋转 左右旋转 顺时针为正

rotateZ：元素围绕其Z轴以给定的读书进行旋转 平面旋转

例如 transform：rotateX(180deg);默认为Z轴

translate 移动

translate（xxpx，xxpx）水平方向和垂直方向同时移动

translateX(xxpx) 仅水平方向移动

translateY（xxpx）仅垂直方向移动

xx可以负数

例如：transform：translateX（12px）；

scale 缩放

scale（xx，xx）使元素水平方向和垂直方向同时缩放

scaleX（xx）元素仅水平方向缩放

scaleY（xx）元素仅垂直方向缩放

xx可以整数和小数

例如：translate；scaleX(1.5)；

skew 扭曲

skew（xxdeg，xxdeg）使元素在水平和垂直方向同时扭曲变形

skewX（xxdeg）仅使元素在水平方向扭曲变形

skewY（xxdeg）仅使元素在垂直方向扭曲变形

例如：translate：skew（12deg，12deg）；

transform-origin 元素的基点位置

认基点就是元素中心位置

transform-origin(xxpx xxpx xxpx)或者（xx% xx% xxpx）
分别为X Y Z轴
其中X和Y的值可以使百分比，em，px，方向

###

transform-style：嵌套元素如何在3D空间中呈现

transform：flat|preserve-3d

flat默认，所有子元素在2D平面呈现

preserve-3d，所有资源在2D平面呈现，设置这个属性之后，就不能overflow：hidden

perspective：视距，视角 设置查看者的位置

当作舞台：多个变形元素的父节点

例如：

.wutai{transform-style;preserve-3d;}

Animation

animation-name 动画属性名

@keyframes 动画关键帧

from to

百分比

例如：@keyframes name{from{width：300px}to{width：600px}}
或者 @keyframes name{10%{}20%{}。。。100%{}}

animation-duration 动画持续时间 单位秒

animation-timing-function 动画效果

属性ease，linear，ease-in，ease-out,ease-in-out,cubic-bezier

animation-iteration-count 指定元素播放动画的循环次数

infinite 无限 数字代表几次

例如：animation-iteration-count:infinite/12;

animation-delay 指定元素动画开始时间

默认值为normal，动画的每次循环都是向前播放 alternate，动画播放在第偶数次向前播放，第奇数次向反方向播放

例如：animation-delay：alternate；

Meta标签

meta提供关于HTML文档的元数据。元数据不会显示在页面上，但是对于机器是可读的，它可用于浏览器（如何显示内容或重新加载页面），搜索引擎（关键词），或其他web服务。

申明编码 meta charset=’utf-8’/>

页面关键字 meta name=”keywords” content=”your tags,”/>

页面描述 meta name=”description”content=”120| 150 words”/>
搜索引擎索引方式

meta name=”robots” content=”index,follow”/>

all：文件将被检索，且页面上的链接可以被查询

none：文件将不被检索，且页面上的链接不可以被查询

index：文件将被检索

follow：页面上的链接可以被查询

noindex：文件将不被检索

nofollow：页面上的链接不可以被查询

html5语义话标签

header 头部

footer 尾部

section 标签定义文档中的节（section、区段）。比如章节、页眉、页脚或文档中的其他部分。

aside 文章的侧栏

nav 导航

article 规定独立的自包含内容，可以包裹section

hgroup 包裹多个h1，h2…

mark 标记 高亮文本常作用于搜素高亮

meter 定义度量衡表示在一定范围内的值，如投票占比，使用量，游戏里的血条等 属性有value配min和max optimum配high和low

progress 定义进度 进度条 属性有value配max和min

time 定义事件和日期 datetime=“YYYY-MM-DDThh:mm:ssTZD”

wbr 定义换行

abbr 定义缩写

cite blockquote q 都是引入的意思 第一个斜体第二个有margin第三个有引号

output 输出 该标签要搭配form和input使用

code 格式化代码

pre 定义预格式化的文本 在pre元素中的文本通常会保留空格和换行符。而文本也会呈现为等宽字体

detail 定义元素的细节 搭配summary（定义细节标题）使用，属性open=“open” summary应为details元素的第一个子元素

dialog 定义对话框 open属性

figure 图表 figcaption 定义图标标题

menu 菜单

main 主要内容

map 用于客户端图像映射，图像映射指带有可点击区域的一幅图像 搭配usemap和area标签使用

address 一般用于footer的地址和电话信息栏非友情链接

下面都是input的属性值

tel 用于输入电话号码

search 用于提示用户输入要搜素的文字 search和text质检的区别主要在于样式上

url 用于输入单个网址

email 用于输入单个电子邮件地址 如果制定了multiple属性，就可以输入多个电子邮件地址，以逗号分隔

datetime 用于输入时区设置韦UTC的日期和时间

date 用于输入不含时区的日期

month 用于输入含年份和月份但不含时区的日期

week 用于输入含年份和周数但不含时区的日期

time 用于输入含小时，分钟，秒和秒的小数部分但不含时区的时间值

datetime-local 用于输入不含时区的日期和时间

number 用户输入数字

range 用于输入数字，但与number的区别在于无需输入具体数字 实施形式大部分为滑块

color 用于通过颜色池控件选择颜色

label 为 input 元素定义标注 搭配for使用 意识就是说可以点框前面的文字选定后面的

datailst 定义选项列表 配合input使用 list属性配合id名

ins 定义已经被插入文档中得文本 del为删除

samp 定义计算机程序的样本文本

上面都是input的属性值

表单属性

autofocus 用于在网页加载后对焦到相关元素上德输入 它的目标可以是输入。选择，文本区域和按钮

placeholder 用于提示用户应输入的数据类型 在对焦到相关元素以及用户输入数据之前，系统会以浅色文字显示占位符值。您可以在输入和文本区域中指定该值。

required 如果设置该属性，它规定用户在提交表单之前必须选择一个值

pattern 属性规定用于验证输入字段的模式 用于身份证号码类型的

autocomplete 自动完成允许浏览器预测对字段的输入。当用户在字段开始键入时，浏览器基于之前键入过的值，应该显示出在字段中填写的选项

autoplay 自动播放

controls 属性规定浏览器应该为视频提供播放控件

loop 单曲循环

preload 提前加载

src 地址

source 单表签 title属性

track 定义用在媒体播放器中得文本轨道 歌词文件 字幕文件

video
